FAQ - Questions Fréquentes

Aventure Projet ? ＋
Ce projet n'es pas abandonnée et n'y sera jamais, je propose d'autre mapping il seront inclus dans le fichier Maps.zip
Est-ce possible d'avoir un patch ? ＋
Merci de ne pas me demander de créer un patch, c'est non ! je ne fournirai jamais de patch.MPQ
Comment installer la map ? ＋
1): Télécharger les fichiers zip
2): Créer un fichier patch-E.mpq dans votre dossier data ensuite placer tout les contenue du fichier MPQ à l'intérieur
3): Ouvrez maps.zip mettez DBCFILESCLIENT et World dans le fichier patch-E.mpq que vous avez créer ensuite lancer le jeu et tester
4): Certaine map possède un sous-dossier il suffit juste de copier par exemple SP dans le fichier world/maps, n'oublier pas d'éditer le fichier map.dbc
Les DBCS ? ＋
Les DBCS servent au bon fonctionnement de la map et incluent des choses comme les noms des zones, les lumières, l'herbe sol roche, soundambience pour les sons custom, MAIS si vous connaissez le modding, vous savez utiliser les DBCS pour votre serveur Créez vos propres lignes pour les adapter Certaines DBC comme créature ne sont pas utilisées à la zone, oui, mais je le laisse car c'était prévu pour autre chose > les sqls optionnel.
SQL OPTIONNEL ＋

Si certain souhaitent les gameobjects, ils sont basé sur des ID custom sur azerothcore. N'hésitez pas à me contacter
Est-ce que c’est compatible avec WoW moderne ? ＋
Seulement la version 3.3.5, à vous de bidouiller pour la version ancienne ou supérieure, je ne fournis aucun soutien.
